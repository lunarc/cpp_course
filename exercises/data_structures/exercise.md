# Exercise: Data structures and file I/O 

Read a list of first and last names from a file, names.txt). Each row of the file contains a first and a last name. The file have unique entries.

Insert the names into a std::map<string, string>, where the first key is the last name and the second key the full name.

Open a second file, sorted_names.txt and write the sorted list of names to this file. Hint: the std::map<> data structures stores its key in sorted order.
